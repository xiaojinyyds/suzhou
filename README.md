# 🏛️ 探索姑苏 - 七座石像文化之旅

一个结合GPS打卡、AI人脸识别、电商点单的苏州文化探索H5网站

## 📸 预览

目前已完成HTML原型图，可以直接在浏览器中打开 `index.html` 查看效果。

### 功能页面预览

- ✅ **欢迎页** - 渐变背景 + 动画效果
- ✅ **石像打卡页** - 7个景点卡片展示
- ✅ **人脸装饰页** - 摄像头预览 + 引导圆圈
- ✅ **甜品点单页** - 7种苏式甜品 + 购物车
- ✅ **我的进度页** - 打卡进度 + 成就系统

## 🚀 快速开始

### 1. 查看HTML原型图

直接双击打开 `index.html` 文件，或使用本地服务器：

```bash
# 使用Python启动服务器
python -m http.server 8000

# 或使用Node.js
npx serve
```

然后在浏览器访问 `http://localhost:8000`

### 2. 查看完整设计文档

打开 `功能设计文档.md` 查看详细的功能设计、技术方案、数据结构等。

## 📋 项目特色

### 🎯 三大核心功能

1. **GPS位置打卡**
   - 基于Geolocation API的精准定位
   - 100米范围内自动打卡
   - 打卡成功展示景点详细介绍
   - 进度追踪和成就系统

2. **AI人脸识别装饰**
   - 使用face-api.js实现实时人脸检测
   - 赛博风格AR装饰效果
   - 引导圆圈提示最佳拍摄位置
   - 一键拍照保存和分享

3. **苏式甜品点单**
   - 7种正宗苏式甜品
   - 完整的购物车系统
   - 实时数量管理
   - 订单统计和历史记录

### 🎨 设计亮点

- **配色方案**：苏州园林主题色（朱红+青灰）
- **动画效果**：淡入、上滑、脉冲等流畅动画
- **响应式设计**：完美适配手机端
- **用户体验**：清晰的视觉反馈和操作提示

## 🛠️ 技术栈（规划）

### 前端
- **框架**：React 18 + TypeScript
- **构建工具**：Vite 4
- **样式**：Tailwind CSS
- **动画**：Framer Motion
- **人脸识别**：face-api.js
- **地图/定位**：Geolocation API + 高德地图

### 后端（可选）
- **服务器**：Node.js + Express
- **数据库**：MongoDB
- **缓存**：Redis
- **存储**：阿里云OSS

## 📖 使用说明

### HTML原型图操作指南

1. **欢迎页**
   - 点击"开始探索"进入主页面

2. **石像打卡页**
   - 点击任意石像卡片查看详情
   - 点击"立即打卡"模拟打卡（实际版本需GPS定位）

3. **人脸装饰页**
   - 查看摄像头预览界面
   - 点击"拍照"模拟拍照（实际版本需人脸检测）

4. **甜品点单页**
   - 点击 + / - 按钮调整数量
   - 点击购物车图标查看已选商品

5. **我的进度页**
   - 查看打卡进度和成就

## 💡 开发建议

### 本地开发
1. 使用HTTPS或localhost（摄像头/定位需要）
2. 准备真实的景点GPS坐标
3. 下载face-api.js模型文件
4. 准备真实的甜品图片

### 性能优化
- 图片使用WebP格式
- 启用代码分割
- 实现懒加载
- 使用CDN加速

### 浏览器兼容性
- Chrome 90+（推荐）
- Safari 14+
- Firefox 88+
- Edge 90+
